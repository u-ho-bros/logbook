# 艦これ 航海日誌+
[航海日誌](http://kancolle.sanaechan.net/)を自分用に改造したものです。
不具合の報告は[こちら](https://github.com/Ibemu/logbook/issues/new)にお願いします。(GitHubの登録が必要です)

## 機能
[本家](http://kancolle.sanaechan.net/) 0.8.1の機能に加えて、

* JSONを保存するオプションの復活
* 受信した全てのJSONを保存
* 任務カウンター
* 艦隊タブに表示する情報の追加
* 大破警告(進撃判定前)
* 出撃報告(戦闘ランク予測)
* 不具合(と思われるもの)の修正
* 他

を実装しています。
